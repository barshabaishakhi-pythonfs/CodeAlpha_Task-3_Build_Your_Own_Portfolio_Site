# CodeAlpha_Task-3_Build_Your_Own_Portfolio_Site

A professional, fully responsive personal portfolio website designed and developed during my Frontend Development internship at **CodeAlpha**. This site acts as a central hub to showcase my skills, featured web projects, resume, and professional contact information.

## 🚀 Features
* **Modern & Responsive Layout:** Tailored to look clean across all screen widths (mobile, tablet, and desktop).
* **Smooth Scrolling:** Fluid navigation transitions between sections (About, Skills, Projects, Contact).
* **Interactive Elements:** Hover states, micro-interactions, and subtle entry animations.
* **Direct Contact:** Accessible links to my professional networks and email.

## 🛠️ Technologies Used
* **HTML5:** Semantic architecture for structured content.
* **CSS3:** Custom layouts, flexbox/grid styling, media queries, and animations.
* **JavaScript (ES6):** For dynamic interactions, mobile menu toggles, and smooth scroll behaviors.
